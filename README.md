# Особенности

Приложение собой представляет таблицу заметок, которые можно как добавлять, так удалять и редактировать. Также приложение может самостоятельно подстроиться под вашу тему устройства. 

# Технологии 

- Swift 5, Xcode 13
- UIKit
- MVC
